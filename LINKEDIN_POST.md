**Headline:** I built a Chrome Extension because I was too lazy to type! 😂

**Body:**

Have you ever paused a YouTube tutorial just to copy some code or text, but realized you can't copy from a video? 😩

I faced this struggle every day. So instead of typing it out manually like a caveman, I decided to build a tool to solve it!

🚀 **Introducing: Text Copier & Offline OCR**

It’s a simple Chrome Extension that lets you:
✅ Extract text from ANY video or image (YouTube, Zoom, etc.)
✅ Enable copy/paste on websites that block it (we all hate those).
✅ **Works 100% Offline!** No API keys, no cloud server, no data stealing.

I built this just for fun using **local OCR technology** (Tesseract.js). It runs an AI engine right inside your browser!

Using "Offline AI" means it might not be as fast as ChatGPT, but it works on a plane, in a cave, or when your WiFi is dead. 🐢⚡️

It was a fun weekend project to learn how to put Machine Learning into a Chrome Extension. The code is open source, so feel free to break it!

**🔗 Check it out on GitHub:** https://github.com/amallpng/text-copier-ocr
